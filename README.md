# 🎀 Blueberry Hampers

A cute & aesthetic hamper ordering website built with HTML, CSS, and JavaScript.

## Features ✨

- 📦 Product catalog with two hamper options
- 📝 Order form with validation
- 💬 WhatsApp integration for orders
- 💾 Order tracking with localStorage
- 🔐 Admin dashboard to view all orders
- 📱 Responsive mobile-friendly design
- 🎨 Beautiful pink & aesthetic UI

## Products 🎁

- **Birthday Hamper** - ₹250
- **Love Hamper** - ₹299
- **Delivery charge** - ₹50

## How to Use 🚀

1. Clone the repository
2. Open `index.html` in a web browser
3. Fill out the order form
4. Click "Pay & Order on WhatsApp"
5. Confirm your order on WhatsApp
